# Chrip

> A WIP project for a simple twitter board

## Install

```
$ git clone git@github.com:adambutler/chirp.git
$ cd chirp
$ bundle install
$ rake db:create
$ rake db:migrate

## Run Specs

```
# With guard
guard

# .. or without
bundle exec rspec
```
